# Evaluación del Caballo de Paso Fino

Spanish-first POC (audio layer only) that ingests a hoof-beat **WAV** recording for a named horse, extracts cadence (BPM) and frequency regularity (coefficient of variation) via pure-JS WAV decode + short-time-energy onset detection (`lib/gaitAnalyzer.js`), applies a deterministic rule engine (`lib/diagnosis.js` — constant-but-lagging → `vet_review`, irregular → `training_adjustment`, else `normal`), persists the evaluation, and shows a diagnostic card + per-horse history.

**Runbook.** Auto-mounted by `src/app.js` at `/evaluacion-del-caballo-de-paso-fino`. Sequelize against `DATABASE_URL` (tables `evaluacion_del_caballo_de_paso_fino_*`, tenant-scoped; falls back to in-memory if the DB is unreachable or `ECPF_FORCE_MEMORY=1`). Writes (horse create, evaluation upload) require a RinglyPro JWT (`JWT_SECRET`); reads are public and tenant-scoped. **Audio: 16-bit PCM mono WAV only** — anything else returns `415`. Video + ML fusion are deferred (multi-year vision); this proves hoof sound is measurable. Run the acceptance suite with `node sit.js` (boots in-memory, generates synthetic WAVs, exits 0 on pass). Privacy: no personal data is processed (Ley 1581 note at `/privacidad`); uploaded audio is analyzed in memory and discarded, and only a generated `evaluation_id` is logged — never the filename. Pages default to Spanish; `?lang=en` switches to English.
