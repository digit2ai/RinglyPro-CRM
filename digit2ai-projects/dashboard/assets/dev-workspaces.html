How this works. Click Open in VS Code to launch VS Code focused on that exact folder (uses the vscode:// handler — works on this Mac only). Once open, start Claude there: your conversation history is stored per-folder, so use /resume in the extension (or click Copy resume cmd for the terminal claude --continue) to pick up where you left off. Each folder also has its own CLAUDE.md, so Claude auto-loads that project's context — no need to remind it to read the repo.
