How This Works

  • Token Flow: Your browser requests a short-lived token from the backend. The API key is never exposed to the browser.
  • WebRTC: Audio streams directly between your browser and ElevenLabs servers - low latency, real-time conversation.
  • Transcript: Speech is transcribed in real-time and displayed as you talk.
  • Backend endpoint: POST /api/elevenlabs-webrtc/token
