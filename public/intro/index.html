How It Works
Como Funciona
  • User intent (chat, voice, webhook) hits Orchestrator
  • La intencion del usuario (chat, voz, webhook) llega al Orquestador
  • Orchestrator selects the right agent based on domain
  • El Orquestador selecciona el agente correcto segun el dominio
  • Agent calls its MCP tools to read/write data
  • El agente llama a sus herramientas MCP para leer/escribir datos
  • Results flow back through the orchestrator to the user
  • Los resultados regresan a traves del orquestador al usuario
