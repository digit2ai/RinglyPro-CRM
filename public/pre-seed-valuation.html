02 — Company Overview

What Digit2AI is

Core product. A multi-tenant CRM with conversational voice AI (agents Rachel, Ana, and Lina), telephony (Twilio), billing and token-metering (Stripe), and a 94-module API surface — deployed in production at aiagent.ringlypro.com

The build engine. "RinglyPro Architect" — an NLP-to-production orchestrator coordinating an 83-agent workforce (8 core + 75 specialists) that turns plain-language briefs into code and deploys via an end-to-end CI/CD loop. This is both a product and the company's structural cost advantage.

Integration fabric. Four Model Context Protocol (MCP) servers (core, GoHighLevel, HISPATEC, OEE) expose tools to AI clients; ElevenLabs conversational agents are provisioned per product.

Go-to-market. Bilingual (EN/ES) throughout, with a partner/champion attribution system, UTM funnel analytics, A/B headline framework, and a voice-driven intake funnel at champion-teaser.html that captures and triages prospect projects in real time.

04 — Valuation Methodology

Three independent lenses

A. Cost / Asset-Replacement Approach Floor

Values the existing software estate by the cost to rebuild it conventionally. The scope — a multi-tenant CRM + voice AI + billing + 22 verticals + ~13 apps + MCP/agent orchestration — represents an estimated 15–30 engineer-years. At a blended fully-loaded cost of $150K–$200K/year, replacement cost is $2.5M–$6.0M. A buyer values the asset, not the discount the AI-native build method achieved.

B. Market-Comparable (Revenue Multiple) Primary at scale

Vertical-AI / SaaS companies transact on forward revenue multiples. Applying current-market ranges to scenario ARR:

ScenarioARRMultipleImplied EV
Pre-revenue / pilots~$0—$3M–$8M

Product-market fit$1.0M6–10×$6M–$10M

C. Strategic / Optionality Approach Upside

Treats the 22-vertical portfolio as a basket of options. Each vertical is a low-cost shot on goal; one reaching PMF re-rates the entire company onto approach B. The AI-native build engine compresses the cost and time of launching each new vertical, raising the expected value of the portfolio relative to a single-product company.

07 — Risk Factors & Value Drivers

What moves the number

Risks / discounts

  • Key-person concentration — authorship is highly concentrated; transferability must be demonstrated.
  • Revenue & retention not yet evidenced — the single largest gate between the $4M and $30M ranges.
  • Breadth-over-depth — 22 verticals can read as a portfolio of POCs rather than one dominant product.
  • Pre-launch / partner-owned IP — some verticals are not yet monetized; AgroMercado IP is held by the ISTC alliance partner.
  • External dependencies — certain features await third-party provider keys and legal review.

Value drivers / upside

  • AI-native build engine — structural cost & speed advantage launching new verticals.
  • Diversified optionality — 22 shots on goal; one PMF re-rates the company.
  • Production deployments — real live systems, not slideware.
  • Bilingual reach — EN/ES across the stack opens LATAM & US-Hispanic markets.
  • Defensible MCP/agent depth — genuine integration fabric, not a thin wrapper.
